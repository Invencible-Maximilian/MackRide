# MackRide 🚗

Sistema de simulação de reservas de viagem desenvolvido em Java.

## 📌 Objetivo

Modelar um sistema com múltiplas entidades aplicando Programação Orientada a Objetos e persistência em arquivos.

## 🛠 Tecnologias Utilizadas

- Java
- Java Collections (ArrayList)
- Manipulação de arquivos
- Tratamento de exceções

## ⚙️ Funcionalidades

- Cadastro de motoristas
- Cadastro de passageiros
- Cadastro de veículos
- Cálculo de custo de viagem
- Persistência de dados em arquivos

## 🧠 Conceitos Aplicados

- Modelagem de Domínio
- Encapsulamento
- Estrutura de Dados
- Regras de Negócio
- Persistência de dados

## 🚀 Como Executar

1. Clone o repositório:
   git clone https://github.com/Invencible-Maximilian/MackRide

2. Compile e execute via IDE Java.

## 📚 Aprendizados

- Organização de múltiplas entidades
- Separação de responsabilidades
- Estruturação de regras de negócio
